# WeKnora Roadmap

이 문서는 WeKnora의 제품 로드맵과 계획 방향을 설명하며 프로젝트 진행에 따라 지속적으로 업데이트됩니다.

## 경량 배포
- [ ] WeKnora가 Embedding, ReRank, LLM, 문서 파싱 등 원자화 호출 API 제공 및 일정 무료 사용량 제공
- [ ] WeKnora가 완전한 클라우드 서비스를 제공하여 플랫폼에서 바로 기능 체험 가능
- [ ] WeKnora Lite 버전 출시로 프라이빗 배포 수요가 낮은 사용자의 빠른 체험 지원

## 지식 이해
- [ ] 전체 문서 파싱 모듈 추상화, 내장 파서/MinerU/기타 파서 전환 지원
- [ ] 문서 청크 전략 최적화, 규칙 기반 외에 의미 기반/장별 청크 지원
- [ ] 문서 구조 시각화: 파싱된 문서의 장 구조, 그래프 관계 등 표시
- [ ] 오디오/비디오 등 더 많은 문서 형식 지원으로 멀티모달 이해 강화

## 검색 및 요약
- [ ] 입력창에서 "@태그"로 검색 범위 지정 지원
- [ ] 입력창에서 이미지/첨부파일 업로드 검색 지원

## 지식베이스 관련 모델 학습
- [ ] 검색 리콜 관련 모델(Embedding, ReRank, LLM 등) 학습
- [ ] 문서 파싱/문서 이해 영역의 지속적 탐색과 자체 모델 고도화

## 지식베이스 형태
- [ ] 지식베이스 형태 확장, 시계열 데이터 저장 및 인덱싱 지원
- [ ] 지식베이스와 Memory 결합 활용 시나리오 탐색

## IM 통합
- [ ] WeCom, Lark 등 IM 시스템과 통합 지원, IM 내에서 WeKnora 기능 사용

## 컴포넌트 및 확장
- [ ] 커뮤니티에서 각 벤더의 모델 서비스/웹 검색 서비스 컴포넌트 유지 관리 장려
- [ ] 지식베이스 관련 Skills 확충에 대한 커뮤니티 기여 장려

## 에코시스템 구축
- [ ] Chrome 확장 제공, "클리핑" 유사 기능으로 웹 콘텐츠를 지식베이스에 저장하고 검색/요약/Q&A 지원
- [ ] 미니 프로그램 플러그인 제공(구체 형태 미정)
- [ ] JS SDK 제공으로 웹에서 WeKnora 기능 통합 용이화
- [ ] 커뮤니티의 VSCode, Cursor, Claude Code 등 에디터/IDE 플러그인 제공 장려

## 문서 구축
- [ ] 공식 문서(사용 가이드, API, 배포 등) 보강
- [ ] 문서/블로그/영상 기여 장려로 커뮤니티 문서 체계 구축
- [ ] Zhihu 플랫폼에서 WeKnora 콘텐츠 모음 구축
